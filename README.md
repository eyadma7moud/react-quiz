<h1 align="center">🧠 React Quiz App</h1>

<p align="center">
  A modern quiz application built with React using <b>useReducer</b> for state management.
</p>

<hr/>

<h2>🚀 Features</h2>
<ul>
  <li>Dynamic questions fetched from API</li>
  <li>State management using useReducer</li>
  <li>Real-time answer feedback (correct / wrong)</li>
  <li>Progress tracking during the quiz</li>
  <li>Final score with highscore system</li>
  <li>Clean and responsive UI</li>
</ul>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li>React.js</li>
  <li>JavaScript (ES6+)</li>
  <li>CSS</li>
  <li>Vite</li>
</ul>

<h2>📂 Project Structure</h2>
<pre>
src/
 ├── components/
 ├── assets/
 ├── App.jsx
 ├── main.jsx
</pre>

<h2>⚡ How It Works</h2>
<p>
The app fetches quiz questions from a local API, then manages all states (questions, answers, score, progress)
using a centralized reducer. Users answer questions, get instant feedback, and see their final results at the end.
</p>
